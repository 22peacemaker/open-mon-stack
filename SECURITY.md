# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/open-mon-stack/open-mon-stack/security/advisories/new). This allows us to assess and address the issue before public disclosure.

Include as much detail as possible: steps to reproduce, affected versions, and potential impact.

We aim to respond within 72 hours and will keep you updated on the remediation timeline.
